# 个人技术笔记仓库

本仓库用于汇总和归档个人在学习过程中整理的技术笔记，涵盖 RAG、Agent、Codex、Git、Docker、强化学习和大模型预训练等多个方向。

---

## 📁 仓库结构

### RAG/
RAG（Retrieval-Augmented Generation，检索增强生成）相关学习笔记，系统梳理了从基础概览到各模块优化的知识：
- `RAG-1-概览.md` — RAG 整体架构与流程概述
- `RAG-2-优化元素提问.md` — 查询优化的策略与方法
- `RGA-3-路由和高级查询.md` — 路由机制与高级查询技术
- `RAG-4-索引优化.md` — 索引结构与优化方案
- `RAG-5-Retrieval检索.md` — 检索阶段的核心技术与策略
- `RAG-6-Generation生成优化策略.md` — 生成阶段的优化方法

### Agent/
Agent（智能体）相关的学习资料与代码实践，分为课程笔记和代码两部分：
- **吴恩达/** — 吴恩达《Agentic AI》课程的学习笔记
  - Agentic 工作流简介、反思设计模式、工具使用、构建 Agentic AI 的实用技巧、高度自主智能体的模式、知识图谱等
- **代码/** — Agent 相关的代码实践与实验记录
  - Jupyter Notebook（上下文工程、可观测性、评估、MCP 互操作性等）
  - Python 脚本（`agent.py`）
  - 评估数据集与日志文件

### codex笔记/
Codex 相关的学习笔记与资料整理：
- `codex.docx`
- `codex_notes_restructured.docx`

### Git与Docker/
开发工具与容器化技术的学习笔记：
- `Docker.md` — Docker 基础概念与常用操作
- `git_note.md` — Git 基础与进阶使用
- `Git操作笔记.md` — 日常 Git 操作备忘

### picture/
各笔记中引用的图片资源文件夹，包含大量 RAG、Agent 等主题的示意图与流程图。
